# 投资理财入门：写给程序员的第一本书

> 面向金融认知为零的程序员，从概念到实操，系统讲清「钱生钱」这件事。

---

## 阅读说明

本教程假设你：
- 完全不懂金融、股票、基金
- 有基本的逻辑思维和信息检索能力
- 希望用最短路径建立正确的投资认知，而非一夜暴富

建议按顺序阅读。每章末尾有小结，章节之间有依赖关系。
