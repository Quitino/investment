## 9.2 财经资讯：哪里看靠谱新闻

| 平台 | 定位 | 特点 |
|------|------|------|
| **财联社** | 及时快讯 | 电报速报，速度快 |
| **东方财富资讯** | 综合财经 | 量大，需筛选 |
| **雪球** | 社区+行情 | 有质量的讨论，也有很多噪音 |
| **第一财经** | 严肃财经 | 深度报道，质量高 |
| **华尔街见闻** | 全球宏观 | 美股、宏观报道翻译快 |
| **Bloomberg/Reuters** | 全球权威 | 英文，第一手 |

**信息质量原则**：
```
一手 > 二手：公告原文 > 媒体解读 > 社交媒体转发
官方 > 个人：交易所公告 > 研报 > 大V分析
数据 > 观点：用数据验证的分析 > 纯观点
```

## 9.3 公司公告与财报：在哪里找一手信息

**A股公司公告（最权威来源）**：
- 上交所：`sse.com.cn`
- 深交所：`szse.cn`
- 巨潮资讯网：`cninfo.com.cn`（最全，一站式）

**财报日历**：
- 年报：4月30日前披露
- 半年报：8月31日前
- 季报：4月/10月底前

**如何快速读财报**：
1. 先看**董事长致股东信**（了解公司战略）
2. 看**利润表**：营收增速、净利润增速、毛利率变化
3. 看**现金流量表**：经营性现金流是否为正
4. 看**管理层讨论分析（MD&A）**：公司自己解释业绩

## 9.7 如何辨别信息质量：一手与二手，信号与噪音

**常见低质信息的识别**：

```
❌ "某只股票明天必涨，赶快买入" → 无依据的价格预测
❌ "内部消息：xx公司要被收购" → 内幕信息（违法）或谣言
❌ "大佬持仓曝光，跟着买稳赚" → 跟风，时机未知
❌ "这只基金今年涨了200%" → 幸存者偏差，不看风险
```

**高质信息的特征**：
- 有数据支撑（财报数字、历史统计）
- 有逻辑链条（原因→推理→结论）
- 有不确定性声明（承认可能错）
- 来源可追溯（公告、研报原文）

> **最重要的原则**：<u>永远要问"他为什么要告诉我这个信息？"。如果有人免费给你"必涨"的信息，他的动机是什么？</u>
