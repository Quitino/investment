# 第十章：开户与实操流程

> 从零开始，手把手完成第一笔投资——以购买沪深300ETF为例。

---

## 10.1 A股开户：券商选择与流程

### 选哪家券商？

对于入门投资者，主要考虑两点：**佣金低** + **App好用**

| 券商 | 佣金（万分之）| 特点 |
|------|-------------|------|
| **华泰证券（涨乐）** | 1.5-2 | App功能全，用户量大 |
| **国金证券（佣金宝）** | 1-1.5 | 佣金极低，适合频繁交易 |
| **中国银河** | 2-3 | 老牌大型券商，稳定 |
| **东方财富证券** | 1.5-2.5 | 和东方财富App深度集成 |
| **招商证券（招招）** | 2-3 | App设计好，功能全 |

> **入门建议**：选华泰或东方财富，开户流程简单，App体验好，佣金合理。不建议追求最低佣金而选择小券商。

### 开户流程（约15分钟）

```
1. 下载券商App
2. 点击"开户"→ 实名认证（身份证两面）
3. 人脸识别 + 视频验证
4. 填写基本信息（职业、收入、投资经验）
5. 风险测评（如实填写）
6. 银行卡绑定
7. 等待审核（通常1个工作日内）
8. 开户成功，入金
```

**注意**：
- 每人只能在一家券商有A股账户（实际是同一个中登账户，可在多家券商开户但同属一个账户）
- 未成年人不能开户
- 开户需要成年人本人操作，不能代办

## 10.2 港股开户：沪港通 vs 港股账户

**方式一：沪港通（推荐入门）**
- 通过你的A股券商账户直接交易港股
- 不需要额外开户，A股开户后即可开通
- 限制：只能买港股通标的（约460支主要股票，含腾讯、美团等）
- 资金：人民币直接买，自动换汇

**方式二：直接开港股账户**
- 通过富途、老虎等港股券商开户
- 可以买所有港股，更灵活
- 需要港元资金（需换汇）

## 10.3 美股开户：富途、老虎、盈透证券对比

| 券商 | 特点 | 适合 |
|------|------|------|
| **富途证券** | 中文界面最好，开户快（最快1天），有模拟盘 | 中国投资者首选 |
| **老虎证券** | 社区功能强，界面友好 | 社交型投资者 |
| **盈透证券（IBKR）** | 费率最低，全球市场覆盖，专业级 | 交易量较大者 |

### 美股开户流程

```
1. 下载App → 填写个人信息（护照）
2. 提交证明文件（护照、住址证明）
3. 风险评估
4. 审核（1-5个工作日）
5. 出入金（国内银行换汇 → 汇款到券商）
```

**换汇注意**：每人每年外汇额度5万美元，大额需银行申报。

## 10.4 基金账户：支付宝、天天基金、券商基金

| 渠道 | 特点 | 费率 |
|------|------|------|
| **支付宝/余额宝** | 操作最简单，货币基金/理财 | 申购费1折 |
| **天天基金（东方财富）** | 基金品种最全，数据好看 | 申购费1折 |
| **各券商App** | 一体化，股票+基金统一管理 | 申购费1折 |
| **基金公司官网/App** | 申购费有时更低（0费率）| 0费率 |

> **推荐**：天天基金品种最全，费率有保障；或者直接在券商App里买，省去切换App的麻烦。

## 10.5 黄金账户：银行App开通

1. 打开工行/建行/招行等银行App
2. 搜索"纸黄金"或"贵金属"
3. 开通贵金属账户（实名认证即可）
4. 最低买0.01克，即可开始

## 10.6 第一笔交易的完整流程（以买入沪深300ETF为例）

### 目标：用1000元买入沪深300ETF（代码：510300）

**Step 1：入金**
```
券商App → 资金 → 银行卡转入 → 输入金额 → 确认
（通常实时到账，最迟T+1）
```

**Step 2：搜索标的**
```
搜索框输入"510300" 或 "沪深300ETF"
查看当前价格（约每手100份 × 每份价格 ≈ 400-500元/手）
```

**Step 3：下单**
```
买入界面：
- 委托方式：限价委托（自己设价格）或市价委托（立即成交）
- 数量：最小1手（100份）
- 价格：参考当前买一/卖一价
→ 确认委托
```

**Step 4：查看持仓**
```
账户 → 持仓 → 看到"510300 沪深300ETF"
显示：持有份数、成本、当前市值、浮盈浮亏
```

**Step 5：卖出**
```
持仓 → 点击该ETF → 卖出
→ 输入卖出数量 → 设置价格 → 确认
（卖出后T+1资金到账，可再买）
```

## 10.7 税务基础：A股免税，美股预扣税

| 市场 | 分红税 | 资本利得税 | 备注 |
|------|--------|-----------|------|
| A股 | 10%（持股不满1年）/ 免税（满1年）| **免征** | 只有印花税0.05% |
| 港股 | 0%（内地居民）| **免征** | |
| 美股 | 30%预扣税（中国居民）| **免征** | 分红被扣30%后才到你账户 |

> **A股资本利得免税**是A股相对其他市场的重要优势——不管你赚多少差价，无需缴税（美国居民买美股要交资本利得税）。
